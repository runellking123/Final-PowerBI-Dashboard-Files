# Final PowerBI Dashboard Files

This repository contains raw data files extracted from Power BI for dashboard analysis.

## Data Files

### Student Data
- `student_master.csv` - Master student information
- `wly_ipeds.csv` - IPEDS student data
- `stud_term_sum_div.csv` - Student term summary by division
- `stud_crs_history.csv` - Student course history

### Academic Data
- `Terms.csv` - Academic term definitions
- `Sections.csv` - Course section information
- `Majors.csv` - Major programs
- `Catalog.csv` - Course catalog
- `j1_degree_history.csv` - Degree history
- `j1_faculty_load_table.csv` - Faculty load information

### Financial Data
- `LineItem.csv` - Financial line items
- `LineItemAmount.csv` - Line item amounts
- `Fees.csv` - Fee information
- `ChargeDefinitions.csv` - Charge definitions
- `ChargeRates.csv` - Charge rates
- `Credit_Types.csv` - Credit type definitions

### Reference Data
- `Year_Definition.csv` - Academic year definitions

## File Format
All files are in CSV format with headers.

## Usage
These files are raw data extracts intended for analysis and reporting in Power BI dashboards.
